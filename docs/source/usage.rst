Usage
=====

.. _installation:

Installation
------------

To use **Protytype Python Library**, first install it using pip:

.. code-block:: console

   (.venv) $ pip install prototype-python-library

Computing fibonacci
-------------------

To compute the n-th element in the _fibonacci sequence_ you can use the ``prototype_python_library.fib()``
function:

.. autofunction:: prototype_python_library.fib

For example:

>>> import prototype_python_library
>>> prototype_python_library.fib(0)
0
